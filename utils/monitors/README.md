# Клієнтські монітори (Vanilla JS)

Джерело скриптів і стилів. Віддаються на тих самих URL, що й раніше:

| Файл | URL |
|------|-----|
| `error-monitor.js` | `/error-monitor.js` |
| `performance-monitor.js` | `/performance-monitor.js` |
| `error-monitor-dashboard.js` | `/error-monitor-dashboard.js` |
| `error-monitor-dashboard.css` | `/error-monitor-dashboard.css` |
| `dev-monitor-panel.js` | `/dev-monitor-panel.js` |
| `dev-monitor-panel.css` | `/dev-monitor-panel.css` |

Маршрути: `app/*.js/route.ts`, `app/*.css/route.ts` → `lib/serve-monitor-asset.ts`.

Редагуйте файли тут, не в `public/`.
